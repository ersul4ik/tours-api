## Tour API
The API provides ability to create/read/delete tour entries

## How to run

1. Make sure you have "Go" installed
2. Create an .env file (on the main.go level)
    - specify DATABASE_URL
2. Run command to build:  `go build main.go`
3. Run: `./main`
